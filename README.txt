Rede Neural para classificar a posição do jogador virtual baseado no dataset FIFA 20.

As posições foram resumidas em:
- Goleiro
- Zagueiro
- Volante
- Meia
- Atacante

Os atributos escolhidos para classificação foram:
- Idade
- Altura
- Peso
- Overall
- Weak Foot
- skill moves
- Shooting
- Passing
- Dribbling
- Defending
- Physics

Como as posições foram alteradas, o dataset modificado usado é o presente neste repositório com nome de "dataset_mod.csv".
Além deste dataset, também está presente neste repositório o dataset do FIFA 19 para realizar os testes.
